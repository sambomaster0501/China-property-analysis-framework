## 🔍 信源验证提醒（必读）

**任何单一信源都不足以构成独立决策的充分依据——这是本框架一以贯之的核心原则。**

在使用本框架进行实际楼盘分析时，请务必做到：

1. **官方优先**：住建局备案价、政府规划文件、教育局学区划片 > 开发商宣传 > 中介说辞
2. **实地验证**：开发商宣传的距离、配套、不利因素，请亲自到场或使用高德/百度地图 API 实测
3. **交叉验证**：同一信息至少通过 2-3 个独立来源确认（如学校对口查询教育局官网 + 区政府公开文件 + 社区公告）
4. **时效检查**：政策、规划、价格信息具有时效性，请确认数据发布时间。本框架所引用的数据与分析，主要基于近三年公开信息——未来市场趋势将随政策与供需格局而演变，无法以既有数据做出确定性的长期保证
5. **样本偏差警惕**：社交媒体上的个案（「这个盘很好/很坑」）不代表整体，关注去化率、投诉量等系统性数据

## 框架定位

对于多数中国家庭而言，不动产购置是生命周期中金额最高、信息不对称最为严重的单次交易。然而，购房者的决策过程往往呈现出自发性与碎片化特征——过度依赖销售话术、被样板间美学引导、以短期市场情绪替代长期价值评估。

本框架的出发点正是补充这一认知缺口。它不是一份推荐名单，而是一套**以经济学理论为底层逻辑、以学术研究为交叉验证的分析方法论**——帮助你在不确定性中建立原则，在噪音中区分信号。

> **授人以渔，而非授人以鱼。框架不预设答案，只构建推导过程。**

## 框架总览：五层漏斗

```
第一层：城市选择 → 评估目标城市是否具备长期持有价值
第二层：板块选择 → 在城市内部甄别配套成熟度与增值潜力兼具的板块
第三层：楼盘选择 → 以 17 维度加权打分体系进行多盘横向对比
第四层：时机判断 → 以 6 盏市场信号灯研判当前是否处于合理买入窗口
第五层：价值监控 → 建立持有期内的持续跟踪与再评估机制
```

各层级之间存在严格的递进关系：上层结论构成下层分析的前提条件，任一环节的判断失误都将传导并放大后续决策的风险。

## 方法论亮点

### 用学术做购房决策

本框架的每个分析模式均植根于学术研究，而非仅凭经验直觉：

| 分析模式 | 理论溯源 |
|---------|---------|
| 「便宜≠好」识别 | CEPR 房价驱动因素研究（2021） |
| 规划兑现评估 | North 制度经济学 + 住建部规划评估办法 |
| 学区溢价判断 | 西南大学 DID 实证研究（10.7万条成交数据） |
| 开发商行为解读 | Spence 信号理论（1973，诺贝尔奖） |
| 市场周期定位 | Howard Marks 周期理论 |
| 压力测试 | SP Global 反向压力测试框架 |
| 决策心理学 | Kahneman 前景理论（1979，诺贝尔奖） |

### 17维度加权打分体系

覆盖价格竞争力、开发商资质、教育交通商业医疗配套、小区品质、得房率、采光噪音、规划落地程度、不利因素排查、市场流通性、宏观政策环境、人口与供需关系、长期持有成本、TOD 价值、交付能力——共 17 个维度，各维度依重要性配置独立权重，配有一套可复用的评分标准。

### 不利因素四分类

将不利因素按噪音源、殡葬设施、工业污染、规划风险四类进行系统化排查，附有距离阈值与影响程度的量化判断依据。
