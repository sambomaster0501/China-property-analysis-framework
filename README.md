## 这是什么？

买房是大多数人一生中最大的一笔消费，但大多数购房者的决策过程却是**凭感觉、听销售、看装修**。

这个框架提供一套**经过学术交叉验证的分析方法论**，帮助你在任何城市、面对任何楼盘时，都能做出理性的购房决策。

### 核心理念

```
不是告诉你要买哪个楼盘，而是教你如何判断一个楼盘值不值得买。
```

---

## 框架总览：五层漏斗

```
第一层：城市选择 → 这座城市值不值得买房？
    ↓
第二层：板块选择 → 城市里哪个板块值得买？
    ↓
第三层：楼盘选择 → 板块里哪个楼盘最好？（17维度打分）
    ↓
第四层：时机判断 → 现在是不是买入时机？（6灯信号灯）
    ↓
第五层：价值监控 → 买入后如何跟踪和判断卖出？
```

每一层都是下一层的前提——城市选错了，板块再好也没用；板块选错了，楼盘再好也亏钱。

## 方法论亮点

### 用学术做购房决策

这个框架不是拍脑袋想出来的。每个分析模式都有学术研究做支撑：

| 分析模式 | 理论溯源 |
|---------|---------|
| 「便宜≠好」识别 | CEPR 房价驱动因素研究（2021） |
| 规划兑现评估 | North 制度经济学 + 住建部规划评估办法 |
| 学区溢价判断 | 西南大学 DID 实证研究（10.7万条成交数据） |
| 开发商行为解读 | Spence 信号理论（1973，诺贝尔奖） |
| 市场周期定位 | Howard Marks 周期理论 |
| 压力测试 | SP Global 反向压力测试框架 |
| 决策心理学 | Kahneman 前景理论（1979，诺贝尔奖） |

### 17维度加权打分体系

覆盖价格、开发商、教育、交通、商业、医疗、品质、得房率、采光噪音、规划落地、不利因素、流通性、宏观政策、人口供需、持有成本、TOD价值、交付能力——共17个维度，每个维度有权重和评分标准。

### 不利因素四分类

系统地排查噪音、殡葬、工业、规划四类不利因素，附距离阈值判断表。

## 🔍 信源验证提醒（必读）

**本框架强调的核心原则是：永远不要盲信单一信源。**

在使用本框架进行实际楼盘分析时，请务必做到：

1. **官方优先**：住建局备案价、政府规划文件、教育局学区划片 > 开发商宣传 > 中介说辞
2. **实地验证**：开发商宣传的距离、配套、不利因素，请亲自到场或使用高德/百度地图 API 实测
3. **交叉验证**：同一信息至少通过 2-3 个独立来源确认（如学校对口查询教育局官网 + 区政府公开文件 + 社区公告）
4. **时效检查**：政策、规划、价格信息具有时效性，请确认数据发布时间
5. **样本偏差警惕**：社交媒体上的个案（"这个盘很好/很坑"）不代表整体，关注去化率、投诉量等系统性数据
