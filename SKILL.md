## 分析哲学：三层漏斗

```
┌─────────────────────────────────────────────┐
│  第一层：城市选择 — 值不值得在这座城市买房？  │  宏观面
│  人口、产业、财政、政策、市场温度、租售比      │
└────────────────────┬────────────────────────┘
                     ↓ 通过
┌─────────────────────────────────────────────┐
│  第二层：板块选择 — 城市里哪个板块值得买？    │  中观面
│  规划兑现率、交通、配套、供需、板块定位         │
└────────────────────┬────────────────────────┘
                     ↓ 通过
┌─────────────────────────────────────────────┐
│  第三层：楼盘选择 — 板块里哪个楼盘最好？      │  微观面
│  17维度加权打分 + 不利因素排查 + 流通性评估     │
└────────────────────┬────────────────────────┘
                     ↓ 通过
┌─────────────────────────────────────────────┐
│  第四层：时机判断 — 现在是不是买入时机？       │  时间面
│  6灯信号灯法 + 宏观周期定位                    │
└────────────────────┬────────────────────────┘
                     ↓ 买入后
┌─────────────────────────────────────────────┐
│  第五层：价值监控 — 持有期如何跟踪和判断卖出？  │  后续面

└─────────────────────────────────────────────┘
```

**核心逻辑**：上层是下层的前提。城市基本面差，板块再好也没用；板块选错了，楼盘再好也亏钱。

## 分析思维内核：拨开迷雾

> **框架是骨架，思维是灵魂。** 以下不是检查清单，是分析思维模型。
> 拿到任何城市、任何楼盘，先想清楚这些问题，再动手查数据。

### 一、信号 vs 噪音（信息分层过滤）

> **理论来源**：Howard Marks《Mastering the Market Cycle》"第二层思维"——超越表面共识，思考二阶效应；ValueGu 信噪比框架——信号是一阶核心数据和高阶逻辑分析，噪音是解释性新闻和碎片化传闻。

市场信息分为三层，分析的第一步是分层过滤。

| 层级 | 类型 | 特征 | 例子 | 处理方式 |
|------|------|------|------|---------|
| **信号** | 基本面变化 | 可量化、可验证、有持续性 | 人口净流入量、GDP增速、库存去化周期、备案价 | 纳入分析框架 |
| **噪音** | 情绪与波动 | 不可量化、短期、反复无常 | "房价要涨了"、"XX板块起飞"、"专家预测" | 过滤掉 |
| **陷阱** | 故意误导 | 有利益方刻意制造 | 隐瞒备案价、"最后一套"、夸大规划距离、虚假名校 | 识别并反向修正 |

**核心原则**：信数据，不信叙事。一个房价涨了10%的故事，不如3个月环比连续收窄的数据有价值。

> **验证来源**：CEPR 学术文献 "What drives house prices"（2021）指出，简单套利理论（房价=未来租金折现值）无法解释房价波动，需关注供需基本面信号而非叙事噪音。

**识别陷阱的四条铁律**（借鉴 ValueGu "24小时冷静期"+"反向思维测试"方法）：
1. 数据来源不可追溯 → 陷阱
2. 结论先于数据（先说"值得买"再找证据） → 陷阱
3. 利益相关方发布（开发商/中介） → 降级处理
4. "据内部消息"/"据业内人士" → 陷阱

### 二、六组分析模式（经理论交叉验证的实战场景）

#### 模式1：「便宜=好」陷阱

> 便宜不是因为被低估，可能是因为被合理定价（甚至高估）。

> **理论来源**：CEPR "What drives house prices"（2021）——房价偏离基本面时，"便宜"可能是市场对未来风险（如不利因素、流动性差）的理性定价，而非错误定价。

**问三个问题**：
1. **为什么这个楼盘比别人便宜？** 位置差？配套差？不利因素？还是市场不认可？
2. **便宜的程度和原因匹配吗？** 如果三环55m只便宜5%，说明市场已消化了这个不利因素，没有额外折价空间
3. **同价位竞品的去化率对比？** 便宜但去化率差 = 市场认为即使折价也不值得

**判断标准**：折价幅度 ≥ 不利因素严重程度 × 2 → 可能是合理的折价；折价幅度 < 不利因素严重程度 → 不是便宜，是有毒。

#### 模式2：「规划兑现」博弈

> 规划是政府的承诺，但政府的承诺不等于未来的现实。

> **理论来源**：North 制度经济学"承诺与可信度"理论——政府承诺的可信度取决于约束机制强度，无约束的承诺价值为零。中国城市规划兑现率研究（住建部《城市总体规划实施评估办法》）——规划调整是常态，远期规划的兑现率通常低于30%。

| 阶段 | 可信度 | 说明 |
|------|--------|------|
| **已建成**（学校开学/地铁开通/商场开业） | 100% | 唯一可信标准 |
| **在建**（工地可见/封顶/装修中） | 70-90% | 仍可能延期 |
| **已获批**（有批文/已招标） | 30-50% | 获批≠开工，开工≠完成 |
| **仅规划**（出现在远期规划文件中） | 10-20% | 规划可以改 |
| **仅口头**（开发商/中介提到） | 0% | 不计入 |

#### 模式3：「学区溢价」评估

> 学区是最大的溢价来源，也是最大的政策风险。

> **实证来源**：西南大学付扬军/曲卫东（2026）《教育均等化措施何以造成"泛学区化"》——基于北京链家107,491条成交数据、339所小学、双重差分模型（DID）的实证研究。核心发现：多校划片政策导致"泛学区化"，普通学区房价反涨2.6%（+1,945元/㎡），但原优质学区房价下降（不显著）。结论：**优质教育资源的稀缺性才是溢价的根本驱动力，政策调整只做价值再分配，不消除溢价。**

**分析公式**：`学区真实价值 = 溢价程度 × 政策稳定性 × 学校历史`

- 老牌名校（>20年）：稳定性高，溢价合理（通常15-30%）
- 新挂牌名校（<5年）：稳定性低，溢价可能是泡沫（多校划片改革可一夜清零）
- 规划名校：溢价=0，风险=无限

**决策原则**（实证支撑）：为学区付溢价前，先评估"如果3年后学区政策变了，这房子还值这个价吗？"——北京案例显示，顶级学区房（宏庙小学）单价从25万/㎡跌至10-12.5万/㎡，跌幅45-60%，验证了政策风险的毁灭性。

#### 模式4：「开发商信号」解读

> 开发商的行为比他们的语言更可信。

> **理论来源**：Spence 信号理论（1973，Nobel 2001）——在市场信息不对称条件下，信号发送方通过"有成本的行动"传递真实信息，接收方通过信号甄别推断隐藏特征。Akerlof "柠檬市场"理论（1970）——信息不对称导致逆向选择。应用到房地产：打折是有成本的信号（牺牲利润），引入名校是无成本的信号（只是揭牌仪式），前者比后者更可信。

| 开发商行为 | 可能含义 | 验证方法 |
|-----------|---------|---------|
| **大力度打折** | 去化压力大 or 现金流紧张 | 查企查查诉讼、债务、员工规模 |
| **频繁加推** | 市场接受度尚可 | 查去化率是否匹配 |
| **长期不推新** | 卖完了 or 市场太差 | 查剩余库存和竞品 |
| **降标减配** | 利润被压缩 | 查合同装修标准条款 |
| **引入名校/名企合作** | 板块本身缺乏吸引力 | 查合作协议是否有政府背书 |
| **销售人员换了一拨** | 内部管理混乱 or 项目有问题 | 查离职原因 |

#### 模式5：「市场周期」定位

> 同一个楼盘，在周期不同阶段买入，收益差距巨大。

> **理论来源**：Howard Marks《Mastering the Market Cycle》——市场在过度乐观与过度悲观之间钟摆，驱动因素是投资者心理、经济基本面、行为偏差三者交织。国信证券"政策预期择时框架"——中国房地产本质是政府"调节周期"的工具，行业基本面越差，政策放松预期越强，反之亦然。

```
繁荣顶部 → 量价齐升，售楼部排队
    ↓ （转折信号：政策收紧/利率上升/供应激增）
衰退初期 → 量缩价挺，开发商仍扛价
    ↓ （转折信号：零星降价/去化率骤降）
衰退中期 → 以价换量，打折促销增多
    ↓ （转折信号：降幅收窄/成交回暖/政策托底）
筑底后期 → 量价低位震荡，政策极度友好
    ↓ （转折信号：环比转正/人口回流/产业升级）
复苏初期 → 量先价后，成交回升但价格仍低
    ↓
新一轮繁荣 → 量价齐升
```

**操作原则**：在筑底后期到复苏初期买入，在繁荣顶部到衰退初期卖出。普通人无法精确择时，但大方向判断并不难——看信号灯。

#### 模式6：「政策预期」博弈（中国特有）

> **理论来源**：国信证券"政策预期择时框架"（2019）——中国房地产本质是政府"调节周期"和"平衡利益"的工具，政策预期是择时的核心变量。

**核心逻辑**：
- 行业基本面越差 → 政策放松预期越强 → 反而是利好
- 行业过热 → 政策收紧预期 → 利空
- 官方舆论风向（人民日报/新华社/住建部表态）是政策预期的先导信号

**分析要点**：
1. 不是看政策本身，而是看**市场对政策的预期**
2. "利好出尽是利空"——政策预期已被price-in时，实际政策落地反而引发回调
3. 跟踪指标：官方舆论松紧风向、信贷额度变化、地方政策试探性松绑

### 三、反向思考（压力测试）

> **理论来源**：Gallagher Mohan 房地产压力测试框架（2024）——通过敏感性分析+情景模拟识别投资脆弱点。SP Global 反向压力测试——从"什么条件下投资失败"出发倒推关键风险因素。

> **对每个推荐结论，必须过一遍反向论证。如果过不了，结论就不可靠。**

**压力测试三问**：
1. **这个推荐，在什么条件下会变成错误决策？**
   - 例："A楼盘值得买" → 如果快速路噪音投诉激增导致加装隔音屏的成本转嫁业主呢？如果新区板块规划5年不兑现呢？如果开发商诉讼导致交房延期呢？
2. **最坏的情况是什么？我能承受吗？**
   - 例：买入后房价再跌10%？开发商延期交房？不利因素导致无法转手？
3. **我忽略了什么？有什么盲区？**
   - 主动寻找反对意见：政务投诉、竞品销售话术、看空分析

**一票否决条件（发现即停止分析，不必再打分）**：
- 开发商有失信被执行记录
- 预售证五证不全
- 土地性质非住宅（40年/50年商业用地当住宅卖）
- 楼栋在地震断裂带上
- 产权有争议或已被抵押

### 四、灵活变通原则

> **框架是指导，不是教条。** 以下情况需要跳出框架，灵活处理。

> **理论来源**：Kahneman & Tversky 前景理论（1979）——人在面对收益和损失时的风险偏好不对称，权重函数呈非线性。"锚定效应"——初始框架会锁定思维，需要主动跳出。Gigerenzer "生态理性"理论——好的决策方法应匹配决策环境的复杂程度，简单环境下复杂模型适得其反。

| 情况 | 框架调整 |
|------|---------|
| 某个维度出现极端值 | 极端值权重自动放大（如不利因素发现殡仪馆<2km，不利因素权重从3%提升至15%） |
| 用户核心诉求极其明确 | 围绕核心诉求重配权重，其余维度降为辅助参考 |
| 市场处于极端状态 | 暴跌中可忽略流通性（都跌），暴涨中加大兑现风险权重 |
| 信息严重不足 | 不猜不估，明确告知"数据不足，建议暂缓"，而非用弱信源硬凑 |
| 两楼盘分数差距<0.2分 | 不应仅凭分数决策，需做深度定性比较和场景分析 |
| 用户只有1个选项 | 不需要打分对比，转为"这个楼盘值不值得买"的单品深度分析 |

### 五、工具与数据调用策略

> 分析不是空想，是数据驱动。遇到分析任务时，主动调用以下工具。

> **方法论基础**：Wickham & Grolemund "Tidy Data"原则（2014）——数据应结构化为观测值×变量的标准格式，来源可追溯。Silver《The Signal and the Noise》——预测的质量取决于信息的质量，GIGO（垃圾进垃圾出）是所有数据分析的底层规律。

| 分析场景 | 推荐工具 | 获取方式 |
|---------|---------|---------|
| 宏观经济/利率/周期 | neodata-financial-search / westock-data | Skill 调用 |
| 地理距离实测 | 高德Web API v3 | 需自行申请免费Key |
| 开发商背景调查 | 企查查/天眼查 | WebFetch |
| 城市统计数据 | 统计年鉴/统计公报/七普 | 政府统计局官网 |
| 房价/成交/土地 | 中指研究院/贝壳研究院/住建局 | 官网/WebFetch |
| 不利因素投诉 | 政务投诉平台（12345/人民网留言板/地方问政平台） | WebSearch |
| 卫星目视排查 | Google Earth / 高德卫星图 | API/Web |
| 板块控规规划 | 城市自然资源和规划局官网 | WebFetch |
| 跨城市对比 | 统一对比口径（房价/租售比/购买力/增速） | 框架第一层 |

**调用原则**：
1. 先用高等级信源，低等级信源仅作补充
2. 关键数据必须 ≥2个独立信源交叉验证
3. 数据冲突时，信源等级高的优先，但标注差异
4. 无法获取数据时，明确标注"数据缺失"而非用弱信源替代

### 六、分析报告输出标准

> **输出不是数据的罗列，是洞察。** 好的分析报告让决策者"看完就知道了"。

**每条数据必须包含**：数据值 + 来源（信源等级） + 验证状态（已验证/待验证/有争议）

**报告结构**：
1. **结论先行**：一句话说清楚买不买、买哪个、为什么
2. **城市判断**：这座城市值不值得投入（1段话）
3. **板块对比**：候选板块的相对优劣（矩阵表）
4. **楼盘打分**：17维度加权结果 + 关键发现（不要只列分数，要解释为什么）
5. **时机建议**：现在该不该出手（信号灯结果）
6. **风险提示**：压力测试结果 + 最坏情况 + 一票否决项排查
7. **行动建议**：下一步具体做什么

**报告避坑**：
- ❌ "根据数据显示" → 必须指明哪个数据
- ❌ "整体来看" → 必须具体
- ❌ "建议关注" → 必须说"关注什么、怎么关注"
- ✅ "住建局备案价2,581,497元（⭐⭐⭐⭐⭐已验证），惠后价2,349,162元，折扣91折，正常范围"

## 第一层：城市基本面分析（六维评估）

> **分析目标**：判断这座城市是否值得投入，以及投入哪个价位段。

### 六维度评估体系

| # | 维度 | 权重 | 核心问题 | 关键指标 | 数据源（通用） |
|---|------|------|---------|---------|---------------|
| 1 | 人口吸引力 | 25% | 人在往哪走？ | 常住人口5年趋势/净增量/青年比例(18-35岁)/高学历占比 | 统计年鉴/七普/人口抽样调查 |
| 2 | 产业竞争力 | 20% | 钱从哪来？ | GDP总量及5年增速/三产占比/千亿产业集群数/上市公司数/高新企业数 | 统计公报/发改委/上市公司协会 |
| 3 | 财政健康度 | 15% | 政府底子厚不厚？ | 一般公共预算收入/土地财政依赖度(<30%健康)/城投债规模 | 财政决算/财政部地方债平台 |
| 4 | 市场温度 | 15% | 市场在升温还是降温？ | 房价指数趋势(环比/同比)/成交面积趋势/库存去化周期/土地流拍率 | 国家统计局70城/中指研究院 |
| 5 | 政策环境 | 15% | 政策是鼓励还是压制？ | 限购限贷限售/首付比例/房贷利率/公积金额度/购房补贴 | 住建局/人民银行/政府公告 |
| 6 | 租售比与流动性 | 10% | 买完租得出去、卖得出去吗？ | 平均租金回报率(>2%及格)/租金收入比(<30%健康)/二手房成交周期 | 贝壳研究院/链家/中指研究院 |

### 城市分级标准

| 等级 | 条件（满足3条以上） | 购房建议 |
|------|-------------------|---------|
| **S级** | 人口持续净增 + GDP增速>全国均值 + 三产占比>60% + 土地依赖<30% | ✅ 核心资产，长期持有 |
| **A级** | 人口净增 + GDP增速≈全国均值 + 有明确产业方向 + 政策友好 | ✅ 可买入，选对板块 |
| **B级** | 人口微增 + GDP增速≈全国均值 + 产业有亮点 | ⚠️ 谨慎，必须精选 |
| **C级** | 人口停滞/流出 + 产业单一 + 土地依赖>40% | ❌ 不建议投资 |

### 分析要点（思维方法，不是检查清单）

1. **人口看增量不看总量**：重庆人口多但增量一般，深圳总量少但增量强
2. **产业看结构不看规模**：资源型城市GDP高但不可持续，科创城市GDP一般但可持续
3. **财政看韧性不看数字**：土地依赖度是关键——<30%说明有造血能力
4. **市场看趋势不看点位**：环比连续3个月收窄比单月涨幅重要
5. **政策看方向不看力度**：政策从紧转松的拐点比力度大小更重要

## 第二层：板块级分析（五因子）

> **分析目标**：在确定的城市中，筛选出值得关注的板块。

### 板块五因子评估

| # | 因子 | 核心问题 | 评估方法 |
|---|------|---------|---------|
| 1 | **规划兑现率** | 承诺的兑现了多少？ | 区分"已建成"/"在建"/"规划中"，只信已建成 |
| 2 | **交通可达性** | 出行方不方便？ | 已开通地铁/快速路，直线距离实测 |
| 3 | **配套成熟度** | 生活需求能否满足？ | 教育(学校)/商业(购物中心)/医疗(综合医院)逐项核实 |
| 4 | **供需关系** | 供大于求还是供不应求？ | 在售新房库存 + 近2年土地供应 + 竞品数量 |
| 5 | **板块定位** | 政府怎么规划这个板块？ | 城市总体规划中的角色(核心/副中心/居住区/产业区) |

### 板块对比矩阵

| 板块 | 规划兑现 | 交通 | 配套 | 供需 | 定位 | 综合评价 |
|------|---------|------|------|------|------|---------|
| A板块 | 80%已建成 | 双地铁 | 齐全 | 供需平衡 | 城市副中心 | ⭐⭐⭐⭐ |
| B板块 | 50%已建成 | 单地铁 | 商业弱 | 供应偏大 | 居住区 | ⭐⭐⭐ |
| C板块 | 从零建设 | 规划中 | 无 | 高供应 | 远期规划 | ⭐⭐ |

### 板块分析核心原则

1. **规划兑现率>规划蓝图**：PPT画得再好，建成才算数
2. **供需关系决定价格走势**：供大于求的板块，价格必然承压
3. **板块定位决定天花板**：政府规划为"居住区"的板块涨幅有限
4. **竞品数量影响流通性**：同价位楼盘<3个最佳，>8个要警惕
5. **注意板块生命周期**：新建期→成长期→成熟期→衰退期

## 第三层：楼盘级分析（17维度）

> **分析目标**：在确定的板块中，精准对比具体楼盘。

### 约束条件（P0一票否决）

| 维度 | 要求 | 性质 |
|------|------|------|
| 预算 | 用户指定，超出即排除 | 硬性 |
| 面积 | 用户指定 | 硬性 |
| 区域 | 用户指定 | 硬性 |
| 预售证 | 必须有，五证不全排除 | 硬性 |
| 开发商 | 央企/国企优先 | 弹性 |
| 楼层/装修 | 用户偏好 | 弹性 |

### 17维度打分体系

| # | 维度 | 权重 | 核心查什么 |
|---|------|------|---------|
| 1 | 价格竞争力 | 15% | 备案价→惠后总价 vs 预算 |
| 2 | 开发商资质 | 10% | 国企/央企/民企，诉讼/处罚/交付先例 |
| 3 | 教育配套 | 9% | 幼儿园→小学→初中完整路径（划片文件核实） |
| 4 | 交通配套 | 8% | 已开通地铁直线距离（规划不算） |
| 5 | 商业配套 | 6% | 已开业购物中心距离 |
| 6 | 医疗配套 | 5% | 综合医院等级和距离 |
| 7 | 小区品质 | 6% | 容积率/车位比/梯户比/物业 |
| 8 | 得房率 | 5% | 含光厅/不含光厅分别计算 |
| 9 | 采光/噪音 | 5% | 噪音源现状 vs 规划区分 |
| 10 | 规划落地 | 4% | 板块内规划已兑现比例 |
| 11 | 不利因素 | 3% | 四分类逐一排查 |
| 12 | 流通性 | 5% | 去化率/竞品数量/不利折价率 |
| 13 | 宏观政策环境 | 5% | 城市政策友好度 |
| 14 | 人口与供需 | 5% | 板块供需关系 |
| 15 | 长期持有成本 | 5% | 月供占收入比/物业/维修 |
| 16 | TOD价值 | 5% | 已开通轨道距离（无轨=1分） |
| 17 | 交付能力 | 4% | 准现房>期房>远期 |

> 详见 `references/scoring.md` 逐维度评分标准

### 不利因素四分类

| 类别 | 典型 | 影响 | 判断 |
|------|------|------|------|
| 噪音类 | 高速/铁路/高架 | 高（持续） | 分贝实测，<500m高危 |
| 殡葬类 | 殡仪馆/火葬场/公墓 | 高（心理） | 区分性质：殡仪馆≠公墓≠咨询处 |
| 工业类 | 变电站/工厂/垃圾站 | 中（间歇） | 核实规划变更 |
| 规划类 | 规划路/高架/市场 | 可变 | 对比控规版本 |

### 流通性评估

| 子维度 | 权重 | 核心指标 |
|--------|------|---------|
| 成交活跃度 | 30% | 板块月均网签量、去化周期 |
| 竞争强度 | 25% | 同价位竞品数量 |
| 折价率 | 30% | 不利因素导致的学术折价（殡仪馆-15~-25%/铁路-8~-15%） |
| 供应冲击 | 15% | 未来2年新增供应 |

### 楼盘分析核心原则

1. **备案价≠宣传价**：永远以住建局备案价为准
2. **已建成≠规划中**：地铁、商业、学校，只算已建成已开业已开通
3. **不利因素距离实测**：高德API直线距离，不信任开发商描述
4. **去化率是市场投票**：同区位去化率低的楼盘，即使数据好看也要警惕
5. **折价率是隐形成本**：不利因素会在出售时体现为低于市场价的成交价

## 第四层：时机判断（6灯信号灯法）

> **分析目标**：确定当前是否为买入时机。

### 6个信号灯

| 信号 | 颜色 | 判断条件 | 数据源 |
|------|------|---------|--------|
| 降幅收窄（≥3月） | 🟢 绿 | 70城房价指数环比降幅连续3个月收窄 | 国家统计局 |
| 成交回暖（≥2月） | 🟢 绿 | 新房+二手房成交量连续2个月环比正增 | 住建局/中指研究院 |
| 货币宽松 | 🟢 绿 | 房贷利率处于近3年最低 | 央行/人民银行 |
| 政策托底 | 🟢 绿 | 国务院或住建部明确"止跌回稳"/"收储"政策 | 国务院/住建部 |
| 同比下跌 | 🟡 黄 | 同比仍在下跌（但环比改善） | 国家统计局 |
| 人口递减 | 🔴 红 | 城市常住人口年增量连续2年下降或负增长 | 统计年鉴 |

### 决策矩阵

| 信号组合 | 建议 | 行动 |
|---------|------|------|
| **4绿+0红** | ✅ 可以入手 | 立即锁定目标楼盘 |
| **3绿+1-2黄** | ⚠️ 谨慎入手 | 看房但不急，等更多信号转绿 |
| **2绿以下** | ❌ 观望 | 下跌趋势未变，保持关注 |

### 时机分析核心原则

1. **环比比同比重要**：同比受基数效应干扰，环比反映真实趋势
2. **连续性>单月数据**：1个月回暖可能是波动，3个月才是趋势
3. **城市分化是常态**：一线城市回暖≠三四线城市回暖
4. **政策拐点>利率绝对值**：政策从收紧转向宽松的拐点比利率具体数值更重要

## 第五层：价值监控（持有期管理）

> **分析目标**：买入后持续跟踪，判断持有还是卖出。

### 持有期关键监控指标

| 指标 | 频率 | 预警线 | 数据源 |
|------|------|--------|--------|
| 板块房价趋势 | 月度 | 连续3个月环比下跌>1% | 贝壳/链家 |
| 板块成交量 | 月度 | 连续3个月环比下降>10% | 住建局/中指 |
| 板块新增供应 | 季度 | 同价位竞品新增>3个 | 土拍公告/住建局 |
| 城市人口数据 | 年度 | 净增量转负 | 统计年鉴 |
| 城市GDP增速 | 季度 | 连续2季度低于全国均值 | 统计局 |
| 贷款利率 | 随时 | 利率处于上升周期 | 央行 |

### 卖出信号（满足2条以上）

1. 板块房价连续6个月环比下跌
2. 板块新增供应集中入市（>5个同价位竞品）
3. 城市人口净流出或增速大幅下降
4. 城市土地财政依赖度超过40%
5. 个人月供压力超过收入的50%

## 信源分级体系（7级铁律）

| 优先级 | 数据来源 | 可信度 | 使用规则 |
|--------|---------|--------|---------|
| 🥇 第1 | **住建局一房一价备案** | ⭐⭐⭐⭐⭐ | 官方备案价，房价分析唯一基准 |
| 🥈 第2 | **用户实地实探数据** | ⭐⭐⭐⭐⭐ | 售楼部一手信息 |
| 🥉 第3 | **政府官网**（gov.cn域名） | ⭐⭐⭐⭐⭐ | 含文号的政府公告 |
| 第4 | **权威官方媒体**（人民日报/新华社/澎湃/第一财经/界面） | ⭐⭐⭐⭐ | 官方授权媒体 |
| 第5 | 有预售证号/公告编号的第三方 | ⭐⭐⭐ | 编号可向政府官网核实 |
| 第6 | **知乎/小红书/哔哩哔哩** | ⭐⭐⭐ | 须满足：转评赞>1000 + 多源3个以上交叉验证 |
| 第7 | 政府信息汇总平台 | ⭐⭐⭐ | 降级使用，标注原始来源 |

> ❌ **百度百科及百度系产品不可作为信源引用**
> ❌ **开发商宣传材料不可作为数据来源**

### 交叉验证原则

```
多源交叉验证（铁律）：
- 关键数据必须 ≥2个独立信源互相支撑
- 信源互斥时 → 深挖修正，不轻信任一单方
- 高德API实测 + 政府文件 + 用户现场 = 三源合一

验证三问（每条数据自检）：
1. 这个数据来自哪里？信源等级是什么？
2. 有没有第二个信源可以佐证？
3. 如果这条数据是错的，结论会怎么变？
```

## 分析组合拳：7步标准流程

### Step 1：明确用户约束
- 预算/面积/区域/城市/自住vs投资
- 硬性约束（P0）vs 弹性偏好
- 特别在意什么（如不利因素/教育/通勤）

### Step 2：城市基本面速评
- 六维度评估 → 城市分级 → 决定是否继续
- 如果C级城市 + 投资目的 → 直接建议放弃

### Step 3：板块筛选
- 五因子评估 → 板块对比矩阵 → 筛选2-3个候选板块

### Step 4：楼盘初筛（排除法）
- 超预算 → 排除
- 无预售证 → 排除
- 五证不全 → 排除

### Step 5：楼盘深度分析
- 建立档案（开发商/价格/配套/不利因素/坐标）
- 不利因素三轮验证
- 配套逐项核实
- 17维度加权打分

### Step 6：宏观与时机判断
- 购入时机信号灯
- 与城市基本面交叉判断

### Step 7：生成分析报告
- 城市基本面结论
- 板块对比结论
- 楼盘打分排名
- 时机建议
- 每条数据标注信源+验证状态

## 跨城市对比方法

> 当用户需要比较不同城市的房产时使用。

### 统一对比口径

| 维度 | 对比方法 | 注意事项 |
|------|---------|---------|
| 房价 | 统一为"每平米单价" | 不要用总价对比（面积不同） |
| 租售比 | 年租金/房价 | 租金取同区域中位数 |
| 购买力 | 房价/人均可支配收入 | 衡量"多少年不吃不喝买得起" |
| 增长潜力 | GDP增速+人口增量+产业升级 | 综合判断，不单看一项 |
| 政策友好度 | 限购/限贷/首付/利率 | 各城市政策不同，需逐项对比 |

### 跨城市决策原则

1. **先比城市再比房**：S级城市+一般楼盘 > B级城市+优质楼盘
2. **核心城市核心区 > 新兴城市优质区**：流动性差异
3. **人口趋势是第一权重**：人走了，什么都没有了
4. **产业决定了"买谁"**：互联网人去深圳/杭州，金融人去上海，制造业人看苏州/东莞
