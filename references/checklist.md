# 楼盘分析必查清单（通用版）

> 每轮楼盘分析必须逐项核实，不得跳过任何条目。适用于所有城市。

---

## 【价格】必查

- [ ] **住建局官网查询备案价**（城市名 + 住建局 → 预/现售项目信息公示）
- [ ] 计算实际单价 = 备案总价 ÷ 建筑面积
- [ ] 对比惠后总价与备案总价，折扣低于85折需高度警惕
- [ ] 查询网签数据与去化率（住建局或房天下）
- [ ] 确认额外费用明细：物业费、车位管理费、装修升级包
- [ ] 确认产权性质（70年住宅 vs 40年公寓）

---

## 【开发商】必查

- [ ] 企查查/天眼查搜索全称 → 检索「失信 / 限制高消费 / 行政处罚」
- [ ] 法律诉讼数量与类型（施工合同纠纷 vs 债务纠纷）
- [ ] 历史名称 → 是否有更名或资产重组记录
- [ ] 已交付项目口碑（政务投诉平台检索）
- [ ] 参保人数（不足50人的小型开发商风险较高）
- [ ] 施工总承包商资质（央企龙头 > 省属国企 > 民营企业）

---

## 【不利因素】必查（三轮验证）

### 第一轮：高德API Text搜索
- [ ] 主要道路（高速 / 快速路 / 高架）→ 直线距离不足500m属高危
- [ ] 殡葬设施 → 殡仪馆 / 殡仪服务点 / 公墓 / 陵园
- [ ] 工业设施 → 变电站 / 工厂 / 垃圾站 / 加油站
- [ ] 记录所有POI名称及其直线距离

### 第二轮：高德API Around搜索（半径5km）
- [ ] 殡葬设施全覆盖
- [ ] 工业设施全覆盖
- [ ] 核实第一轮POI是否遗漏

### 第三轮：政务投诉平台交叉核实
- [ ] 搜索楼盘名称 → 噪音投诉 / 工厂投诉
- [ ] 搜索具体POI名称 → 性质核实（是否已搬迁或关闭）
- [ ] 殡葬类：区分「殡仪馆（遗体火化）」与「公墓（仅骨灰安葬）」与「咨询处（销售点）」

---

## 【地铁/交通】必查

- [ ] 城市轨道交通官网查询线路状态（仅计入**已开通**线路）
- [ ] 高德API实测**直线距离**
- [ ] 判断标准：
  - <800m：⭐⭐⭐⭐⭐ 步行可达
  - 800-1200m：⭐⭐⭐⭐ 骑行可达
  - 1200-2000m：⭐⭐ 距离偏远
  - >2000m：⭐ 基本不可用
- [ ] ⚠️ 规划中地铁**不计入**，仅已开通线路有效

---

## 【学区】必查

- [ ] **区教育局官网**查询年度划片文件（须为PDF格式且有文号）
- [ ] 确认幼儿园：是否公立或普惠性
- [ ] 确认小学 → 初中完整升学路径（不应仅关注小学）
- [ ] 区分：**已开学学校** 与 **规划中 / 待建**
- [ ] 核实近三年划片是否保持稳定
- [ ] ⚠️ 销售人员口头承诺的学区归属一律无效

---

## 【医疗】必查

- [ ] 高德API实测最近**综合医院**距离
- [ ] 确认医院等级（三甲 > 二甲 > 二乙）及科室设置
- [ ] ⚠️ 必须区分综合医院与专科医院（妇幼医院不接诊全科患者）
- [ ] 判断标准：<1km 优秀 / 1-2km 良好 / >2km 偏差

---

## 【商业】必查

- [ ] 高德API实测最近已开业购物中心距离
- [ ] ⚠️ 区分：**已开业** vs **在建** vs **规划中**
- [ ] 判断标准：<1km 步行可达 / 1-2km 骑行可达 / >2km 生活不便

---

## 【小区品质】必查

- [ ] 容积率：<2.0 优秀 / 2.0-2.5 良好 / >3.0 偏差
- [ ] 车位比：≥1:1.2 优秀 / 1:1~1:1.2 良好 / <1:1 偏差
- [ ] 梯户比：1T2 最优 / 2T2 良好 / 2T3 可接受
- [ ] 物业：检索口碑（政务投诉平台搜索）
- [ ] 绿化率：≥35% 优秀 / 30-35% 良好 / <30% 偏差

---

## 【不利因素快速判断阈值】

| 设施 | 高危（慎重） | 中等 | 可接受 |
|------|------------|------|--------|
| 高速/快速路/高架 | <500m | 500-1000m | >1000m |
| 铁路 | <800m | 800-1500m | >1500m |
| 殡仪馆 | <5km | 5-8km | >8km |
| 公墓/陵园 | <2km | 2-5km | >5km |
| 变电站 | <500m | 500-1000m | >1000m |
| 工厂 | <1km | 1-3km | >3km |
| 垃圾站 | <1km | 1-2km | >2km |
| 加油站 | <300m | 300-800m | >800m |
| 殡葬咨询处 | — | — | 可忽略（非实体墓地） |

---

## 【交房与产权】必查

- [ ] 确认交房时间（精确至楼栋、批次、日期）
- [ ] 确认产权性质（70年住宅 vs 40年商业）
- [ ] 查验预售证号（住建局官网核实）
- [ ] 延期交房违约金条款
- [ ] 装修标准是否明确写入合同
