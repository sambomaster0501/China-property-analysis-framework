## 案例复盘：方法论沉淀（8条核心经验）

1. **备案价必须查**：住建局备案价是房价分析的唯一基准，开发商宣传价不可信
2. **去化率是市场投票**：去化率低说明市场不认可，即使数据好看也要警惕
3. **不利因素必须有距离**：不能说"附近有铁路"，必须说"铁路直线1.33km"
4. **殡葬要区分性质**：殡仪馆（遗体火化）≠ 公墓（骨灰安葬）≠ 咨询处（销售点）
5. **板块从零建设 ≠ 板块成熟**：从零建设型板块配套需时间兑现，成熟板块配套已就绪
6. **得房率含赠送/不含赠送必须分开**：86% vs 97%差距巨大，赠送空间计入与否改变判断
7. **首秀项目 = 风险点**：开发商此前无住宅交付先例，品质和工期不可预期
8. **实际距离必须实测**：地理编码坐标存在偏差（某案例：宣传"步行商业"实际4.0km）；距离必须用地图 API 实测，不信任开发商宣传数据

## 入选方法论原则（从本案例提炼）

1. **先看备案价再看惠后价**：开发商给出的参考价和折扣前，先去住建局查备案价
2. **核心不满足就排除，不勉强**：超预算、不在主城区、非住宅用地，直接排除
3. **不利因素三重确认**：实地实探 + 卫星地图 + 公示牌三重交叉验证
4. **配套只算已建成**：规划中的地铁/学校/商场不算配套
5. **不靠记忆，靠记录**：每次分析结果记录下来，不靠下次翻聊天记录
