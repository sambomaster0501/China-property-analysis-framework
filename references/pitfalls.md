# 常见陷阱清单（通用版）

> 来自实战复盘，踩过的坑必须记住。适用于任何城市。

---

## 一、价格陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| 折扣后"优惠价"高于备案价 | 住建局官网查备案价，对比惠后总价 | 多付冤枉钱 |
| 备案价被隐瞒 | 城市+住建局官网查楼盘备案均价 | 实际贵10%+ |
| 去化率低但仍在售 | 住建局网签数据或房天下，去化率<40% | 市场不认可 |
| 装修升级包强制购买 | 问清是否有升级包，价格多少 | 隐形成本 |
| 车位价格未告知 | 问清车位价格，是否必须购买 | 额外支出 |
| 物业费虚高 | 对比同区域同类产品 | 长期持有成本 |
| "XX万起"是最低价房源 | 核实具体房号，确认目标户型价格 | 被低价引流 |

## 二、开发商陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| 国企背景但有大量诉讼 | 企查查搜索开发商全称 | 交房可能延期/质量问题 |
| 首次操盘无交付先例 | 企查查查开发项目数量 | 品质不可预期 |
| 改名/资产重组规避债务 | 企查查历史名称搜索 | 历史问题未解决 |
| 项目违规变更规划 | 政务投诉平台搜索开发商+项目名 | 已购业主权益受损 |
| 虚假国企背景 | 核实实际股权穿透至最终控股方 | 非真正国企背书 |
| 物业口碑差 | 政务投诉平台搜索"物业投诉" | 入住后服务差 |

## 三、学区陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| "名校就在旁边"但未签约 | 要求出示合作办学协议原件 | 学校不来，承诺落空 |
| 规划学校充实现有 | 查政府官网学校是否已开工 | 等了几年学校没来 |
| 小学强但初中弱 | 查完整小升初划片路径 | 初中学区差 |
| 划片范围每年变化 | 查近3年划片文件对比 | 买了房但划不进 |
| 销售口头承诺学区 | 要求书面承诺（录音） | 事后无法追责 |
| 不在主城区学区内 | 确认行政区域归属 | 无法享受主城教育资源 |

## 四、交通陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| 规划中地铁算作已开通 | 只信城市轨道交通官网已开通状态 | 入住后几年无地铁 |
| "TOD上盖"但步行>1km | 高德API实测，不要信开发商 | 实际不是TOD盘 |
| 高架/铁路"远期减少噪音" | 不信承诺，只信现状和文件 | 入住后噪音依旧 |
| 高速噪音被低估 | 高德API直线距离<500m必查 | 夜间无法开窗 |
| 地铁口距离实测不准 | 用高德API步行模式实测 | 宣称800m实际1500m |

## 五、不利因素陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| **公墓=殡仪馆**（最大陷阱） | 殡仪馆含遗体火化；公墓仅骨灰安葬 | 殡仪馆干扰远大于公墓 |
| 殡葬设施「咨询处」= 实体墓地 | 高德around大量"陵园咨询处"是销售点 | 虚惊一场 |
| 山体/建筑阻隔被忽略 | 核实墓地方向与山体位置关系 | 心理压迫感不同 |
| 加油站"已搬迁"但项目在搬迁前 | 查政府公告搬迁时间 | 入住后加油站才搬 |
| 变电站"规划中"不在意 | 查城市控规最新版本 | 建成后才知道 |
| 工厂/市场"已关闭"有延时 | 核实关闭时间与入住时间关系 | 入住后仍在运营 |
| 竞品不利因素被忽视 | 对候选楼盘做相同标准的不利因素排查 | 做了A但没做B，对比失真 |

## 六、户型陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| 宣传得房率≠实际得房率 | 住建局备案合同面积核实 | 实际套内缩水 |
| 入户光厅算入得房率 | 问清光厅是否计入产权面积 | 实际得房率低于宣称 |
| 阳台/设备平台赠送不写入合同 | 确认赠送空间是否写入合同 | 赠送空间无保障 |
| 次卧面积极小（<8㎡） | 查房间面积是否满足基本使用 | 只能做书房 |
| 户型图顶底不等（异形） | 核实顶面和底面尺寸是否一致 | 家具摆放受限 |
| 电梯井紧邻卧室 | 查户型图楼层平面关系 | 低频噪音干扰 |
| 北向主卧 | 查主卧朝向 | 冬季日照差 |

## 七、合同陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| 延期交房无违约金 | 检查合同条款 | 开发商无压力 |
| 违约金过低 | 行业惯例为已付房款万分之一/日 | 维权成本高 |
| 装修标准不写入合同 | 要求装修明细写入合同附件 | 货不对板 |
| 车位产权不明确 | 问清是人防车位/产权车位/租赁 | 无法办理产权 |
| 公积金贷款被拒后强制商贷 | 提前确认公积金贷款条件 | 利率增加 |
| 交房标准与样板间差异 | 合同注明"与样板间一致" | 货不对板 |

## 八、分析过程陷阱

| 陷阱 | 识别方法 | 后果 |
|------|---------|------|
| 只看价格，不看备案价 | 住建局核实 | 被开发商宰 |
| 只看地铁，不分已开通/规划 | 只信已开通 | 入住后无地铁 |
| 只看小学，不看初中学区 | 查完整路径 | 初中差 |
| 不利因素只搜一次 | 必须Around全覆盖 | 遗漏周边设施 |
| 信了开发商口头承诺 | 必须书面+政府文件 | 承诺落空 |
| 用安居客数据不核实 | 企查查/政府官网交叉验证 | 数据失真 |
| 第三方自媒体数据当官方 | 户型面积等必须官方文件 | 数据偏差 |
| 只做数据罗列不做分析 | 输出必须有洞察和结论 | 没有决策价值 |
| 机械套框架不看上下文 | 根据实际情况灵活调整权重和方法 | 分析僵化 |
| 只找支持结论的证据 | 主动寻找反对意见做压力测试 | 结论不可靠 |
| 地理编码坐标不准 | 用 POI 直接搜索（非地理编码）获取精确坐标 | 距离偏差1.8km+，导致分析全错 |
| 距离引用开发商宣传数据 | 高德/百度 API 实测步行+驾车双距离 | 宣传0.5km实际4km（真实案例） |
