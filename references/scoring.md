# 评分标准参考

> 每个维度1-5分，5分最优。打分必须附理由和信源。

---

## 评分语义定义

| 分数 | 含义 | 使用场景 |
|------|------|---------|
| **5分** | 优秀，明显领先 | 在候选中明显优于其他楼盘 |
| **4分** | 良好，满足需求 | 满足用户主要需求，有小缺陷 |
| **3分** | 一般，可接受 | 有缺陷但在可接受范围内 |
| **2分** | 较差，有硬伤 | 有明显问题，需特别关注 |
| **1分** | 很差，慎重 | 严重问题，核心需求无法满足 |

---

## 各维度评分参考

### 1. 价格竞争力（权重15%）

| 分数 | 条件 |
|------|------|
| 5分 | 完全在预算内，余地≥10% |
| 4分 | 在预算内，余地5-10% |
| 3分 | 在预算内，余地<5% |
| 2分 | 超出预算≤5% |
| 1分 | 超出预算>5% |

### 2. 开发商资质（权重10%）

| 分数 | 条件 |
|------|------|
| 5分 | 央企/国企，无诉讼，无处罚，多个成功交付先例 |
| 4分 | 国企/央企，无重大诉讼，有交付先例 |
| 3分 | 国企/民企龙头，有少量诉讼但已解决 |
| 2分 | 民企，有较多诉讼或处罚记录 |
| 1分 | 有失信记录，或首次操盘无交付先例 |

### 3. 教育配套（权重9%）

> 综合评估幼儿园→小学→初中全链条

| 分数 | 条件 |
|------|------|
| 5分 | 该城市顶级初中（如北京四中/上海华二/深圳中学等），有政府文件确认对口 |
| 4分 | 较好初中，已开学，口碑好；小学已确定 |
| 3分 | 一般初中，多校划片摇号；小学已确定 |
| 2分 | 弱校或规划中学校（未开学），学区不确定 |
| 1分 | 无对口初中或极差学校 |

### 4. 交通配套（权重8%）

| 分数 | 条件 |
|------|------|
| 5分 | 双地铁已开通，直线<800m |
| 4分 | 已开通，直线800-1200m |
| 3分 | 已开通，直线1200-2000m（骑行可接受） |
| 2分 | 已开通，直线>2000m |
| 1分 | 无已开通地铁，规划中不计入 |

### 5. 商业配套（权重6%）

| 分数 | 条件 |
|------|------|
| 5分 | 已开业购物中心<1km，步行可达 |
| 4分 | 已开业购物中心1-2km，骑行可达 |
| 3分 | 已开业购物中心2-3km |
| 2分 | 无已开业商业，依赖规划中 |
| 1分 | 周边无商业配套规划 |

### 6. 医疗配套（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 三甲综合医院<1km，含急诊 |
| 4分 | 三甲或二甲综合医院1-2km |
| 3分 | 二乙/社区医院2-3km |
| 2分 | 附近仅专科医院，不接诊全科 |
| 1分 | 无医疗配套 |

### 7. 小区品质（权重6%）

| 分数 | 条件 |
|------|------|
| 5分 | 容积率<2.0，车位比≥1:1.2，1T2或2T2，物业好 |
| 4分 | 容积率2.0-2.5，车位比≥1:1，2T2，物业中上 |
| 3分 | 容积率2.5-3.0，车位比1:0.8-1:1，2T3，物业一般 |
| 2分 | 容积率>3.0，车位比<1:0.8 |
| 1分 | 容积率>4.0，开放式或有严重品质问题 |

### 8. 得房率（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 得房率≥100%（新规+光厅），赠送空间丰富 |
| 4分 | 得房率95-99%，有赠送空间或光厅 |
| 3分 | 得房率85-94%，无赠送 |
| 2分 | 得房率<85% |
| 1分 | 得房率<80%或户型有严重缺陷 |

### 9. 采光/噪音（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 无噪音源，采光无遮挡 |
| 4分 | 有轻微噪音源（>500m），或规划中有噪音源但已设缓冲带 |
| 3分 | 中等噪音源500-1000m，或已采取降噪措施 |
| 2分 | 高速/铁路/高架500m内 |
| 1分 | 高速/铁路<100m，噪音严重超标 |

### 10. 规划落地（权重4%）

| 分数 | 条件 |
|------|------|
| 5分 | 板块规划100%已落地，即买即住 |
| 4分 | 主要规划已落地（>80%），少量在建 |
| 3分 | 部分规划已落地（50-80%），3-5年可成熟 |
| 2分 | 少量规划落地（<50%），5-8年可成熟 |
| 1分 | 从零建设，8年以上 |

### 11. 不利因素（权重3%）

> 分数越高代表不利因素越少（越好）

| 分数 | 条件 |
|------|------|
| 5分 | 5km内无不利因素 |
| 4分 | 存在1类轻微不利因素，距离较远 |
| 3分 | 存在1-2类不利因素，在中等距离 |
| 2分 | 存在2-3类不利因素，较近，或1类严重不利 |
| 1分 | 存在3类以上不利因素+殡仪馆<3km，或铁路<100m |

### 12. 流通性（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 板块低供应+高去化+无竞品+无不利折价（A级流通性） |
| 4分 | 板块供需平衡+去化正常+少量竞品（B级） |
| 3分 | 板块供应较大+去化一般+有竞品（C级） |
| 2分 | 板块高供应+去化慢+竞品多+有折价因素（D级） |
| 1分 | 板块供应过大+大量竞品+严重折价（E级） |

### 13. 宏观政策环境（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 城市定位国家级，政策全面利好 |
| 4分 | 城市定位省级/区域级，政策偏利好 |
| 3分 | 政策中性，无重大利好或利空 |
| 2分 | 政策偏紧，有限购/限贷等限制 |
| 1分 | 政策严厉，交易成本极高 |

### 14. 人口与供需（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 城市人口持续高增，板块供不应求 |
| 4分 | 城市人口正增，板块供需平衡 |
| 3分 | 人口微增，板块供应适中 |
| 2分 | 人口停滞，板块供应偏大 |
| 1分 | 人口净流出，板块严重过剩 |

### 15. 长期持有成本（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 月供<30%收入，物业费低，无大修预期 |
| 4分 | 月供30-40%收入，物业费适中 |
| 3分 | 月供40-50%收入，物业费偏高 |
| 2分 | 月供>50%收入，持有压力大 |
| 1分 | 租不抵贷，月供严重超出承受能力 |

### 16. TOD价值（权重5%）

| 分数 | 条件 |
|------|------|
| 5分 | 双轨TOD步行盘（<800m），板块发展明确受轨道驱动 |
| 4分 | 双轨步行盘（800-1200m）或单轨TOD步行盘（<800m） |
| 3分 | 单轨步行盘（800-1500m） |
| 2分 | 单轨骑行盘（>1500m） |
| 1分 | 无已开通轨道 |

### 17. 交付能力（权重4%）

| 分数 | 条件 |
|------|------|
| 5分 | 准现房（<6个月），国企/央企背书，施工进度正常 |
| 4分 | 近期交付（6-12个月），国企/央企，施工进度正常 |
| 3分 | 期房（12-18个月），国企/央企 |
| 2分 | 期房（18-24个月），民企 |
| 1分 | 期房>24个月，开发商有烂尾/延期先例 |

---

## 权重自定义规则

```
用户约束条件 → 权重调整

死守预算 → 价格权重 +5%
非常介意不利因素 → 不利因素权重 +3%
非常看重教育 → 教育权重 +3%
依赖地铁通勤 → 交通+TOD权重各+3%
投资型 → 流通性权重 +5%，教育权重 -3%
自住刚需 → 交付能力权重 +2%
```

---

## 加权总分计算

```
加权总分 = Σ(各维度得分 × 权重)

满分 = 5.00分

参考解读：
- 4.5+：优秀楼盘，强烈推荐
- 4.0-4.4：良好楼盘，重点考虑
- 3.5-3.9：一般楼盘，有明显取舍
- 3.0-3.4：较弱楼盘，需谨慎
- <3.0：较差楼盘，不推荐
```
